Graylog released version 7.1.3, a bug-fix update designed to enhance platform functionality. The release includes multiple installation options such as DEB and RPM packages, Docker container images, and tarball distributions for both open and enterprise versions. Users are directed to upgrade notes and changelogs for detailed information about improvements and can report issues through the GitHub issue tracker.
